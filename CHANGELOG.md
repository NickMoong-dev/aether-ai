# Changelog

## [v1.1.0] - 2026-03-06

### 추가
- 초기 릴리즈
- AI 소설 집필 도우미 (Ollama 기반 LLM)
- ComfyUI 기반 캐릭터/커버 이미지 생성
- 백과사전 RAG (nomic-embed-text 임베딩)
- 에피소드 편집기, 캐릭터 관리, 세계관 사전
- 자동 백업 및 수정 이력
- Electron 자동 업데이트 (electron-updater)
- GitHub Actions CI/CD 파이프라인
